Summary: Latitude vs. X

The purpose of this project is to analyze the weather changes as you move closer to the equator. I pulled data from the OpenWeatherMap API and created a list of over 500 cities.

I plotted charts through Matplotlib that compared Latitude vs. Weather. The factors that were compared are max temperature, humidity, cloudiness and wind speed. This site is a visualization of the data in addition to an analysis of any trends found in the comparisons.
